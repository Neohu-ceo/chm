Codebase Health Monitor — 完整文档

📦 安装

pip install chm-cli        # 基础 CLI
pip install chm-cli[mcp]   # 含 MCP 支持

需要 Python 3.9+ 和 Git

🚀 快速开始

$ chm demo          # 无需真实项目，立即体验
$ chm init .        # 在项目中初始化
$ chm analyze .     # 完整 8 维分析
$ chm doctor .      # 诊断 + 处方

🔍 chm analyze

运行全部 8 个分析维度，输出健康度评分和详细报告。

chm analyze .                      # 终端输出
chm analyze . --report html -o r.html  # HTML 报告
chm analyze . --report json -o r.json  # JSON 输出
chm analyze . -n 1000              # 分析最近 1000 个提交

🩺 chm doctor

智能诊断——分析所有维度，输出优先级排序的、可执行的改进建议。

chm doctor .

🔌 chm mcp — AI Agent 集成

将 CHM 作为 MCP 服务器运行，AI 代理可以直接查询代码库健康度。

chm mcp                  # stdio 模式 (Claude Desktop)
chm mcp --http 8080      # HTTP 模式
claude mcp add chm -- chm mcp

12 个 MCP 工具 + 1 个资源

🔐 API 认证

POST /api/auth/register   {"email","password","name"}
POST /api/auth/login      {"email","password"}
GET  /api/auth/me          当前用户信息
POST /api/auth/logout

🔑 API Keys

GET    /api/keys            列出所有 Key
POST   /api/keys            创建新 Key (返回一次!)
DELETE /api/keys/:id        吊销 Key

# CLI 认证
chm login YOUR_API_KEY
# 或环境变量
export CHM_API_KEY=chm_xxx
