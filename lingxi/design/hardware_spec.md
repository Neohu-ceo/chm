# 灵犀 Lingxi — 硬件规格书

> 基于外观设计图自动分析生成

## 外观分析结果

- 造型: 圆形/圆角方形角色
- 色调: 暖色系（肤色/奶油色）
- 特征: 眼睛、面部轮廓
- 复杂度: 中等（6个主要轮廓）
- 触感: 预设柔软/硅胶材质

## LED 灯光方案

| 位置 | 用途 | LED 类型 |
|------|------|---------|
| 面颊区域 | 情绪表达（腮红发光） | WS2812B ×2 |
| 眼睛区域 | 闪烁/眨眼效果 | 小型 LED ×2 |
| 底部 | 氛围灯/充电指示 | WS2812B ×4 |
| 头顶 | 通知/互动提示 | 单色暖光 LED |

## 触控交互

| 区域 | 手势 | 触发 |
|------|------|------|
| 头顶 | 轻拍 | 唤醒/问候 |
| 面部 | 抚摸 | 亲密度+ |
| 身体 | 握住 | 进入倾听模式 |
| 底部 | 双击 | 切换模式 |

## 灵犀后端 API

设备通过 WiFi 连接后端：

```
POST /api/bond          — 设备绑定
POST /api/chat          — 发送语音转文字，返回回复+灯光+振动
POST /api/hardware/heartbeat — 心跳，返回当前情绪状态
```
