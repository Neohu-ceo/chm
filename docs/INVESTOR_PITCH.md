## 问题

工程团队每年在代码质量工具上花费数十亿美元，但现有方案有三个根本缺陷：

1. **太重**: SonarQube 需要独立服务器、复杂配置、专人维护
2. **太贵**: CodeClimate 等 SaaS 方案按座位收费，大团队成本失控
3. **不透明**: 闭源产品让你不知道评分是怎么算出来的

**结果**: 多数中小团队干脆不用任何代码质量工具。

---

## 解决方案

Codebase Health Monitor (CHM) — 一个命令就够了：

```bash
chm analyze .
```

- **零配置**: 不需要服务器、不需要注册、不需要配置文件
- **本地运行**: 代码从不离开你的机器
- **Git 原生**: 直接读取 Git 历史，不侵入代码
- **开源核心**: CLI 工具永久免费，MIT 协议
