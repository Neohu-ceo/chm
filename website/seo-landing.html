常见问题

这和 SonarQube 有什么区别？

SonarQube 是代码静态分析（检查代码写法），CHM 是代码演化分析（检查代码改动模式）。两者互补。而且 CHM 不需要服务器，一个 pip install 就好。

我的代码会被上传吗？

不会。CHM 只读你的 Git 历史（git log），所有分析在你的机器上完成。代码永远不会离开你的电脑。

能分析多大的仓库？

默认分析最近 500 个提交。对于 10,000+ 提交的仓库，分析时间仍然在 1 秒以内。你可以用 -n 参数调整范围。

支持哪些语言？

CHM 是语言无关的——它分析 Git 历史而非代码语法。复杂度分析目前支持 Python、JavaScript、TypeScript、Go、Rust、Java 等 15+ 种语言。
