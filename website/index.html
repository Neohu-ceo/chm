功能

你的代码库，一目了然

不需要服务器、不需要配置文件、不需要 API key —— 一条命令就够了。

🔥

代码热区检测

自动识别被频繁修改的文件。高流失文件往往是架构问题的信号——重构的第一目标。

👥

贡献者分析

了解谁在写什么、知识如何分布。巴士因子警告让你在关键人物休假前有所准备。

💓

团队脉动

24 小时活动分布、提交节奏、活跃日统计。一眼看清团队的开发习惯和健康度。

🧩

复杂度估算

识别高复杂度、低注释率的高风险文件。不需要 AST 解析，轻量级但有效的启发式分析。

📊

HTML 可视化报告

一键生成精美的自包含 HTML 报告。色彩编码的健康度仪表盘，可以直接分享给团队或管理层。

🔒

本地优先，数据不出门

一切分析在本地完成。不需要上传代码到云端，不需要联网，不需要注册账号。
